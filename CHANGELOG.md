# Changelog

All notable changes to **Legal Case Researcher** will be documented in this file.

## [1.0.0] — 2025-07-15

### Added
- Core legal research engine with Gemma 4 via Ollama
- Case law research with jurisdiction filtering
- Legal precedent finder with relevance scoring
- Legal argument strength analyzer
- Case citation extraction from documents
- Case document summarizer
- CLI interface with Rich formatting
- Streamlit web UI with dark theme
- FastAPI REST API with OpenAPI documentation
- Docker and Docker Compose support
- Sample legal research scenarios
- Comprehensive test suite with mocked LLM
- Privacy-first architecture — all data stays local
- Configuration via YAML and environment variables

### Security
- Zero external API calls — 100% local processing
- No telemetry or data collection
- Attorney-client privilege fully protected
